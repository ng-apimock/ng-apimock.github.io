Makes sure that [@ng-apimock/core](https://github.com/ng-apimock/core) will reset the mocks back to the default settings.

## Usage
```typescript
resetMocksToDefault();
```
Executing this will reset: 
- the selected scenario's
- the delays
- the echo indicators
